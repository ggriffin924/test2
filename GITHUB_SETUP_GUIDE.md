# 🌐 Creating a New GitHub Repository (Web UI)

Follow these steps to create a clean, empty repository for your project.

### 1. Log in to GitHub
Go to [github.com](https://github.com) and sign in to your account.

### 2. Create a New Repository
*   Click the **"+"** icon in the top-right corner of any page.
*   Select **"New repository"**.

### 3. Configure Your Repository
*   **Repository name:** Type `gemini-portfolio`.
*   **Public/Private:** Keep it **Public** (required for the free tier of GitHub Pages).
*   **Initialize this repository:**
    *   **CRITICAL:** Leave **"Add a README file"** UNCHECKED.
    *   **CRITICAL:** Leave **"Add .gitignore"** set to **None**.
    *   **CRITICAL:** Leave **"Choose a license"** set to **None**.
    *   *We want a completely empty "blank slate" so the Gemini CLI can handle the initialization.*

### 4. Create Repository
*   Click the green **"Create repository"** button at the bottom.

### 5. Copy Your HTTPS URL
*   You will be taken to a "Quick setup" page.
*   Ensure the **"HTTPS"** tab is selected (not SSH).
*   Click the **Copy icon** next to the URL (it looks like `https://github.com/YOUR_USERNAME/gemini-portfolio.git`).
*   **Keep this URL handy!** You will paste it into the Gemini CLI in the next step.
