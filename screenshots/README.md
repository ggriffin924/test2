# 📸 Project Screenshots

This folder is the designated location for all project screenshots. 

### How to Take and Save Screenshots

#### 🪟 Windows (PC)
1.  **Capture:** Press `Windows Key + Shift + S` to use the Snipping Tool.
2.  **Save:** 
    *   Select the area you want to capture.
    *   Click the notification that appears in the bottom-right corner.
    *   Click the **Save** icon (floppy disk) and navigate to this `screenshots/` folder.
    *   *Alternative:* Press `Alt + Print Screen` to capture your active window, then paste it directly into a graphics editor to save here.

#### 🍎 macOS (Mac)
1.  **Capture:** Press `Command + Shift + 4` to select an area or `Command + Shift + 5` for more options.
2.  **Save:** 
    *   By default, macOS saves screenshots to your Desktop.
    *   **Drag & Drop:** Simply drag the screenshot file from your Desktop and drop it into the `screenshots/` folder inside your VS Code sidebar.
    *   *Pro Tip:* You can change your default screenshot location by pressing `Command + Shift + 5`, clicking **Options**, and selecting **Other Location...** to point directly to this folder.

---
*Note: Keeping your screenshots organized here helps the Gemini CLI understand and document your project's visual progress!*
